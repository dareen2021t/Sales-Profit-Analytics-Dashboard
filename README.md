



# 📊 Sales & Profit Analytics Dashboard



## Overview

This project presents an interactive Power BI dashboard designed to analyze sales and profit performance across products, customers, regions, and time. It provides a clear view of business performance to support strategic, data-driven decisions.

## Key Features

 Sales Performance Analysis
 Profitability Breakdown
 Top & Bottom Products
 Customer Segmentation
 Regional Insights (Region, State, City)
 Time-Based Trends (Monthly & Quarterly)

## Key Insights

* A small number of products drive most of the revenue
* Some customers contribute significantly to overall profit
* Regional performance varies strongly across states and cities
* Seasonal trends impact both sales and profitability
* Not all high-sales products are highly profitable


## Tools & Technologies

* Power BI
* DAX
* Data Modeling
* Data Cleaning


## Business Value

This dashboard helps stakeholders:

* Identify top-performing and underperforming areas
* Improve pricing and product strategy
* Understand customer behavior
* Support data-driven business growth decisions
